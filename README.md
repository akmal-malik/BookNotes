## Tech Stack
- Node.js
- Express.js

- EJS

## Features
- Add, view, and delete books
- Store notes and ratings
- Persistent database storage
- Fetch book covers via ISBN
- Clean responsive UI

## How to Run
1. Clone the repo
2. npm install
3. Create MySQL database using provided schema
4. Add .env file
5. npm run dev
6. Open http://localhost:3000
